Appendix — Design Notes

This site, annotated.

Every layout choice in this site maps to a CPSC 349 principle. Hover the orange ◆ tags scattered across the pages for in-context explanations, or read the full catalog below.

Heuristic 01 · Nielsen
Visibility of System Status
Where: rail pulse, hover states, form focus, breadcrumb.
The pulsing "Markets Open" indicator, every hover, and the active-page highlight in the nav all keep the user oriented at all times.
Heuristic 03 · Nielsen
User Control & Freedom
Where: breadcrumbs, prev/next page nav, multi-route navigation.
Every page has a breadcrumb back to Home plus explicit Previous/Next links at the bottom. The user is never stuck.
Heuristic 04 · Nielsen
Consistency & Standards
Where: shared stylesheet, repeated section headers, identical nav.
A single styles.css drives all 5 pages. Every page follows "§ number — title" headers and the same monospace label conventions.
Heuristic 05 · Nielsen
Error Prevention
Where: application form.
Required fields marked with an accent asterisk, inline hints below ambiguous fields, sensible input types (email, select). Errors prevented before they occur.
Heuristic 06 · Nielsen
Recognition over Recall
Where: nav with labels, form select dropdowns, breadcrumbs.
Users see options rather than having to remember them. Dropdowns list pod choices; nav is always visible; breadcrumbs show the path.
Heuristic 08 · Nielsen
Aesthetic & Minimalist Design
Where: ruthless palette, two type families.
Two type families, four colors, one button style. Every extra element competes for attention with the content that matters.
Cognitive · Miller
Miller's Law (7 ± 2)
Where: hero stat strip, nav, pillars.
Headline stats capped at 4, pillars at 4, nav at 5. Each grouping sits comfortably below the working-memory ceiling.
Cognitive · Hick–Hyman
Hick's Law
Where: navigation, CTAs.
Reaction time grows with the log of choices. A 5-item nav lets a visitor decide where to go in under a second.
Motor · Fitts
Fitts's Law
Where: nav links, buttons, coverage cells, page-nav blocks.
Click targets are oversized — nav has 12px vertical padding, buttons are 14×28px, page-nav blocks span half-page width. Targets are large and near where eyes land.
Perception · Gestalt
Proximity & Common Region
Where: pillars block, coverage grid, footer.
Elements that belong together are physically grouped and share a bounding rule. The eye reads structure before it reads words.
Perception · Gestalt
Similarity
Where: team cards, process dots, pod tags.
Visual sameness signals categorical sameness. Three identical cards = three peers. A row of identical dots = a sequence.
Norman · Affordances
Signifiers & Feedback
Where: button shadows, card lifts, cursor changes, focus rings.
Every interactive element looks interactive. Every action returns visual feedback within 100ms (hover lift, color change, focus ring).
